# Shivra Tech - Tech Solutions Website

Professional React/Next.js website for Shivra Tech providing:
- AI/ML Solutions
- Web Development
- Android Development
- Cloud Database Solutions
- Custom Projects

## Getting Started

### Prerequisites
- Node.js 16+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Building for Production

```bash
npm run build
npm start
```
